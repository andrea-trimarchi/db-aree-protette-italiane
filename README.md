# db-aree-protette-italiane
Il progetto consiste in una progettazione concettuale e logica di una base di dati MySQL per la gestione delle aree protette italiane. È stato svolto per conseguire l'esame di Sistemi Informativi Aziendali 1 del corso di laurea magistrale Amministrazione e Gestione Digitale delle Aziende.



**Gestione Aree Protette Italiane** - **Progetto di Database**

Progetto d'esame per il corso di Sistemi Informativi Aziendali I (LM-77, Università degli Studi di Torino, a.a. 2025/2026). Voto ottenuto: 27/30 (progetto + interrogazione orale).

**Contenuto del progetto:**

Progettazione completa di una base di dati per la gestione delle aree protette italiane, comprendente progettazione concettuale (schema E-R), progettazione logica (schema relazionale, analisi delle ridondanze, tavola dei volumi e delle operazioni) e implementazione in MySQL tramite phpMyAdmin.
